# laminas-mvc-skeleton

## Install

```bash
git clone git@github.com:wwwjsx/sencha-laminas.git
```

### Install dependencies

```bash
composer install
```

### Run application

```bash
composer server
```

```bash
http://localhost:8085
```

### Sencha application

```bash
/public/sencha/app
```

### Sencha production build

```bash
/public/build
```

### DB SQLite 3

```bash
/data/riit.db
```
